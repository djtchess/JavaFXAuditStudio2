# Frontend Angular

## Cible technique

- Angular `21.2.4`
- TypeScript `5.9.x`
- RxJS `7.8.x`

## Verification des versions

Verification realisee le `16 mars 2026`.

Sources officielles :

- `https://angular.dev/reference/releases`
- `https://angular.dev/reference/versions`

Constat :

- Angular `21` est la version en support actif ;
- la ligne `v21.2` est planifiee semaine du `23 fevrier 2026` ;
- la documentation officielle est construite en `v21.2.4`.

## Usage

```bash
npm install
npm run start
```

Le frontend appelle le backend via `/api` avec `proxy.conf.json`.
